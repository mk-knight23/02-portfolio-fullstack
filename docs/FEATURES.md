# Features

## Search & Filter
- Real-time search by project name or technology
- Filter by status: Live, Error, Skipped
- Instant results with smooth animations

## Stats Dashboard
- Total project count
- Live deployments count
- Target tech stack indicator
- Blocked/error count

## UI Design
- Glassmorphism cards with blur backdrop
- Animated background blobs
- Gradient text effects
- Hover lift animations

## Project Cards
- Project number badge
- Status indicator (Live/Error/Skipped)
- Technology tags
- Direct links to live demo and GitHub

## Accessibility
- Skip to main content link
- Keyboard navigable
- Proper focus states
- Semantic HTML

## Performance
- Code splitting via Vite
- Optimized animations
- Lazy component rendering
