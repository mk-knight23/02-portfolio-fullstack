# Architecture: 02-repos-browser

## Overview
A dynamic, high-density project explorer built with React 19 and Vite 6. This application serves as a central hub for browsing and filtering the modernized repository ecosystem of mk-knight23.

## Tech Stack
-   **Framework**: React 19
-   **Language**: TypeScript
-   **Styling**: Tailwind CSS v4
-   **Build Tool**: Vite 6
-   **UI Patterns**: Component-based architecture with reactive state management.
-   **Motion**: Framer Motion for layout transitions and micro-interactions.

## Key Components
-   **Dashboard**: Main layout containing stats cards and project grid.
-   **ProjectCard**: High-fidelity project display with status indicators and launch links.
-   **StatCard**: Visual summaries of modernization progress.
-   **Filters**: Real-time search and status-based filtering system.

## Performance
-   Optimized React 19 concurrent features.
-   Lightning-fast assets delivery via Vite.
-   Low-latency filtering and layout transitions.
