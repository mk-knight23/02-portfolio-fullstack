# Design System: 02-repos-browser

## Visual Philosophy
The design focuses on **high-density information display** without overwhelming the user. It utilizes a premium dark aesthetic with vibrant accents.

## Design Patterns
-   **Grid Layout**: Flexible and responsive grid for displaying project cards.
-   **Glassmorphism**: Subtle transparency and backdrop-blur for a modern, depth-focused feel.
-   **Visual Hierarchy**: Clear distinction between project status, tech stack, and action buttons.

## Color Palette
-   **Primary**: Deep Slate (#0f172a / #020617)
-   **Accent 1**: Violet (#8b5cf6)
-   **Accent 2**: Fuchsia (#d946ef)
-   **Status Colors**: Green (Live), Red (Error), Yellow (Skipped)

## Interactivity
-   **Hover States**: Scale and glow effects highlight active projects.
-   **Filtering**: Instant UI feedback as users search or toggle status.
-   **Motion**: Staggered entrance animations for a polished load experience.
